# 七孔探针校准算法

## 1. 概述

七孔探针是一种用于测量三维流场的气动探针，包含 7 个压力孔：
- **6 个周向孔**（p1~p6，间隔 60° 均匀分布）
- **1 个中心孔**（p7）

通过 7 个压力值反算来流的**总压 Pt**、**静压 Ps**、**来流角度 (α, β)** 和**速度 V / 马赫数 Ma**。

算法采用**双模式切换**策略：
- **小角度模式**：来流角在 ±30° 以内，6 孔对称处理
- **大角度模式**：来流角超出 ±30°，选取压力最大的两个孔分区域处理

---

## 2. 探针孔位布局

```
         p1 (0°)
           ▲
     p6   / \   p2
   (300°) |p7| (60°)
     p5   \ /   p3
   (240°)  ▼  (120°)
         p4 (180°)
```

| 孔号 | 周向位置 | 说明 |
|------|----------|------|
| p1 | 0° | 前缘孔 |
| p2 | 60° | |
| p3 | 120° | |
| p4 | 180° | 背风孔 |
| p5 | 240° | |
| p6 | 300° | |
| p7 | 中心 | 中心孔（参考孔） |

## 3. 输入数据格式

### 3.1 压力数据（单次测量）

```
{
  "p1": 压力值1,  "p2": 压力值2,  "p3": 压力值3,
  "p4": 压力值4,  "p5": 压力值5,  "p6": 压力值6,
  "p7": 压力值7,  "t":   温度(℃), "pa": 大气压(Pa)
}
```

### 3.2 校准文件格式（.prb 文件）

```
标头行（忽略）
ka  kb  cpt  cps  a  b
...（每行 6 个字段，空格分隔）
```

- **小角度校准文件**：`7.prb` — 12×12 网格，a∈[-30°,30°], b∈[-30°,30°]，步长 5°
- **大角度校准文件**：`1.prb` ~ `6.prb` — 每个孔对应的扇区校准，a∈[30°,45°], b∈[0°,355°]

## 4. 主流程

```
FUNCTION 七孔探针校准(压力数据, 校准文件目录):
    1. 读取压力数据 {p1~p7, t, pa}
    2. 计算小角度 ka, kb
    3. 读取 7.prb 校准文件，构建边界多边形
    4. 射线法判断 (ka, kb) 是否在小角度边界内
    5. IF 在边界内 (sign == 0 或 1):
         → 小角度模式处理
    6. ELSE:
         → 大角度模式处理
    7. 计算速度和马赫数
    8. 返回 {α, β, Pt, Ps, V, Ma}
```

### 模式判定逻辑

```
小角度模式：点 (ka, kb) 落在 7.prb 校准数据围成的多边形内部或边界上
大角度模式：点 (ka, kb) 在边界外
```

## 5. 小角度模式

小角度模式基于所有 6 个周向孔的压力值，采用对称化处理计算（ka, kb）坐标，然后在校准网格中插值得到角度和压力系数。

### 5.1 流程

```
FUNCTION 小角度模式(压力数据, 校准数据):
    1. 计算 ka, kb（小角度对称公式）
    2. 构建 12×12 校准网格（a∈[-30,30], b∈[-30,30], 步长 5°）
    3. 在网格中定位点 (a, b) 所在的四边形
    4. 距离反比插值 → 精确 (a, b)
    5. 双线性插值 → 系数 (cpt, cps)
    6. 解析求解 → 总压 Pt, 静压 Ps
    7. 返回
```

### 5.2 小角度 ka, kb 计算公式

```
p_avg = (p1 + p2 + p3 + p4 + p5 + p6) / 6

cpa = (p4 - p1) / (p7 - p_avg)
cpb = (p5 - p2) / (p7 - p_avg)
cpc = (p6 - p3) / (p7 - p_avg)

ka = (cpb + cpc) / √3
kb = -(2·cpa + cpb - cpc) / 3
```

### 5.3 小角度校准网格构建

```
构建 144 个四边形（12×12 网格）:
  a ∈ [-30, -25, ..., 25, 30]  (12 个值, 步长 5°)
  b ∈ [-30, -25, ..., 25, 30]  (12 个值, 步长 5°)

每个四边形顶点:
  X1(a, b)        X2(a+5, b)
      ┌──────────┐
      │          │
      │   (a,b)  │
      │          │
      └──────────┘
  X4(a, b+5)      X3(a+5, b+5)

从校准文件匹配每个顶点的 (ka, kb) 值
```

### 5.4 四边形内插值 (KaKb → a,b)

对于每个四边形的 4 条边，计算直线方程 k=j·ka+b_j：

```
边1 (X1→X2): k2 = (kb2-kb1)/(ka2-ka1), b2 = kb1 - k2·ka1
边2 (X2→X3): k3 = (kb3-kb2)/(ka3-ka2), b3 = kb2 - k3·ka2
边3 (X3→X4): k4 = (kb4-kb3)/(ka4-ka3), b4 = kb3 - k4·ka3
边4 (X4→X1): k1 = (kb1-kb4)/(ka1-ka4), b1 = kb4 - k1·ka4
```

判断点是否在当前四边形内：
```
y1 = k1·ka + b1    (左边界在 ka 处的 kb)
y3 = k3·ka + b3    (右边界在 ka 处的 kb)
x2 = (kb - b2)/k2  (上边界在 kb 处的 ka)
x4 = (kb - b4)/k4  (下边界在 kb 处的 ka)

条件: y1 ≤ kb ≤ y3  AND  x4 ≤ ka ≤ x2
```

若在四边形内，距离反比插值：
```
d1 = |(-k1)·ka + kb - b1| / √(k1²+1)   // 到边1距离
d2 = |(-k2)·ka + kb - b2| / √(k2²+1)   // 到边2距离
d3 = |(-k3)·ka + kb - b3| / √(k3²+1)   // 到边3距离
d4 = |(-k4)·ka + kb - b4| / √(k4²+1)   // 到边4距离

b = b_X1 + d1/(d1+d3) · 5    // b 角度插值
a = a_X1 + d4/(d2+d4) · 5    // a 角度插值
```

### 5.5 双线性插值 (a,b → cpt,cps)

定位 (a,b) 在校准网格中的四边形，该四边形 4 个顶点都已知 (a,b,cpt,cps)：

```
对 cpt 和 cps 分别计算:

沿 b 方向（上边 X1→X2，下边 X3→X4）:
  k_上 = (c_X2 - c_X1) / (a_X2 - a_X1)
  c_上 = k_上 · a + (c_X1 - k_上 · a_X1)

  k_下 = (c_X4 - c_X3) / (a_X4 - a_X3)
  c_下 = k_下 · a + (c_X3 - k_下 · a_X3)

沿 a 方向插值:
  c = (c_下 - c_上) · (b - b_X1) / 5 + c_上

最终得到: cpt, cps
```

### 5.6 Pt, Ps 解析计算

已知 cpt, cps 及原始压力值，解方程组：

```
cpt = (p7 - Pt) / (Pt - Ps)
cps = (Ps - p_avg) / (Pt - Ps)

解析解:
  denom = 1 + cpt + cps
  Pt = (p7·(1 + cps) + cpt·p_avg) / denom
  Ps = (cps·p7 + p_avg·(1 + cpt)) / denom
```

## 6. 大角度模式

当来流角超出 ±30° 范围时，采用大角度模式。该模式选取压力最大的前两个孔作为候选，分别尝试计算。每个候选孔对应一个独立的校准文件 (`N.prb`)。

### 6.1 流程

```
FUNCTION 大角度模式(压力数据, 校准文件目录):
    1. 找到 p1~p6 中压力最大的两个孔号 (first, second)
    2. 以 first 孔为中心计算 ka, kb
    3. 检查 (ka,kb) 是否在 first 孔的校准边界内
    4. IF 在边界内:
         → 用 first 孔的校准数据完成计算
    5. ELSE:
         → 以 second 孔为中心计算 ka, kb
         IF 在边界内:
             → 用 second 孔的校准数据完成计算
         ELSE:
             → 越界处理（边界外推）
    6. 坐标变换 θ,φ → a,b
    7. 返回
```

### 6.2 大角度 ka, kb 计算公式

以孔 n 为中心（n ∈ {1..6}）：

```
p_center = p{n}
p_left   = p{n-1}  (n=1 时 p_left=p6, 向左循环)
p_right  = p{n+1}  (n=6 时 p_right=p1, 向右循环)

denom = p_center - (p_left + p_right)/2

ka = (p_center - p7) / denom
kb = (p_left - p_right) / denom
```

### 6.3 最大压力孔选取

```
找出 p1~p6 中压力最大的两个孔:
  pressures = [(1, p1), (2, p2), ..., (6, p6)]
  按压力降序排列
  first_hole  = 最大压力孔号
  second_hole = 次大压力孔号
```

### 6.4 大角度校准网格构建

大角度校准区域：a ∈ [30°, 45°]，b ∈ [0°, 355°]

以孔 n 为中心，构建 36（12×3）个四边形：
```
a 范围: 30°, 35°, 40°, 45°  (3 个区间)
b 范围: 孔 n 中心 ±30°，步长 5°  (12 行)

b 起始角度: 60·n - 30
b 递减: b_start - 5·j  (j=0..11)
b 角度归一化: 若 < 0 则 += 360
```

边界处理：
```
当 b=0 时，b-5 不是 -5 而是 355（角度周期 360°）
```

### 6.5 边界多边形构建

每个孔的校准数据构建 4 条边界线：
```
边界0: b=孔右侧边缘, a=30,35,40,45  (4个点)
边界1: a=45°, b遍历12个角度值         (13个点)
边界2: b=孔左侧边缘, a=30,35,40,45  (4个点, 反转)
边界3: a=30°, b遍历12个角度值         (13个点, 反转)
```

### 6.6 Pt, Ps 解析计算（大角度）

```
以孔 n 为中心:
  p_center = p{n}
  p_left   = 左邻孔压力
  p_right  = 右邻孔压力
  p_side   = (p_left + p_right) / 2
  p_avg    = (p1+p2+p3+p4+p5+p6) / 6

  cpt = (p_center - Pt) / (Pt - Ps)
  cps = (Ps - p_side) / (Pt - Ps)

解析解:
  denom = 1 + cpt + cps
  Pt = (p_center·(1 + cps) + cpt·p_side) / denom
  Ps = (cps·p_center + p_side·(1 + cpt)) / denom
```

### 6.7 坐标变换 (θ,φ → a,b)

大角度模式下，校准数据中的 a 表示 θ（俯仰角），b 表示 φ（方位角）。
最终输出需要转换为标准的 (a, b) 角度：

```
保存原始值:
  θ_result = a
  φ_result = b

边界保护: 若 |θ| > 89.5°, 跳过转换（tan 趋于无穷）

θ_rad = radians(θ)
φ_rad = radians(φ)

tan_a = tan(θ_rad) · sin(φ_rad)
tan_b = tan(θ_rad) · cos(φ_rad)

a_output = -degrees(arctan(tan_a))
b_output =  degrees(arctan(tan_b))
```

## 7. 越界处理（边界外推）

当 (ka, kb) 点既不在小角度边界内，也不在大角度两个候选孔边界内时：

```
FUNCTION 越界处理:
    1. a 角度设为校准边界最大值 45°
    2. 根据 kb 值在边界线上线性插值 b 角度:
       IF kb 超出边界范围:
           b = 孔 n 的中心角度 (0,60,120,...)
       ELSE:
           在边界线的 kb 值序列中插值 b 角度
    3. 以边界值 (a=45, b) 继续后续计算
    4. 返回
```

## 8. 射线法多边形判断

用于判断 (ka, kb) 点是否在校准数据边界多边形内。

```
FUNCTION 射线法(点坐标, 多边形顶点列表):
    从点向右发射水平射线
    统计射线与多边形边的交点数量

    交点判断规则:
      - 射线从下方穿过边: toggles inside
      - 点在边上: 返回 0（边界上）
      - 奇数个交点: 返回 1（内部）
      - 偶数个交点: 返回 -1（外部）

    特殊处理:
      - 水平边（y1 == y2 == y）: 检查点是否在边上
      - 顶点相交: 只统计 y 在 (min_y, max_y] 范围内的情况
      - 右边界的点: 计入交点
```

伪代码：
```
function point_in_polygon(point, polygon):
    x, y = point
    inside = False
    on_boundary = False
    n = len(polygon)
    (p1x, p1y) = polygon[0]

    for i in 1..n+1:
        (p2x, p2y) = polygon[i % n]

        // 点在水平边上
        if y == p1y and y == p2y:
            if min(p1x, p2x) <= x <= max(p1x, p2x):
                return 0  // 边界上

        // 射线穿过边
        if min(p1y, p2y) < y <= max(p1y, p2y):
            if x <= max(p1x, p2x):
                if p1y != p2y:
                    x_inters = (y - p1y) * (p2x - p1x) / (p2y - p1y) + p1x
                    if p1x == p2x or x <= x_inters:
                        inside = not inside

        (p1x, p1y) = (p2x, p2y)

    return 1 if inside else -1
```

## 9. 速度与马赫数计算

```
FUNCTION 计算速度马赫数(压力数据, 结果数据):
    输入: t(温度℃), pa(大气压 Pa), Pt(总压), Ps(静压)
    常数: R = 287.06 J/(kg·K), γ = 1.4

    T_k = t + 273.15                           // ℃ → K
    ΔP  = |Pt - Ps|                             // 动压

    速度: V = √( 2 · ΔP · R · T_k / pa )       // m/s

    压力比 = (Pt + pa) / (Ps + pa)
    马赫数: Ma = √( 5 · |(压力比)^(0.4/1.4) - 1| )

    边界检查:
      - 若 Pt < Ps: 数据异常警告（总压不应低于静压）
      - 若压力比 < 0: 测量异常警告
```

## 10. 完整算法伪代码

```
FUNCTION 七孔探针校准_主函数(压力JSON字符串, 校准文件目录):

    // === 第0步: 数据加载 ===
    data = 解析JSON(压力JSON字符串)
    {p1,p2,p3,p4,p5,p6,p7,t,pa} = data

    // === 第1步: 小角度 ka,kb 计算 ===
    p_avg = (p1+p2+p3+p4+p5+p6) / 6
    cpa = (p4-p1) / (p7-p_avg)
    cpb = (p5-p2) / (p7-p_avg)
    cpc = (p6-p3) / (p7-p_avg)
    ka = (cpb + cpc) / sqrt(3)
    kb = -(2*cpa + cpb - cpc) / 3

    // === 第2步: 加载小角度校准 ===
    cal_7prb = 读取PRB文件(校准目录 + "/7.prb")
    // cal_7prb: [{ka,kb,cpt,cps,a,b}, ...]

    // === 第3步: 构建小角度边界 ===
    边界多边形 = 构建小角度边界(cal_7prb)
    // 4条边: b=-30, a=30, b=30, a=-30

    // === 第4步: 判断模式 ===
    sign = 射线法((ka,kb), 边界多边形)

    IF sign == 0 OR sign == 1:  // 在边界内或边界上
        // ===== 小角度模式 =====
        网格 = 构建12x12四边形网格(cal_7prb)
        (a,b) = 四边形内插值_KaKb((ka,kb), 网格)
        四边形 = 定位a_b所在四边形((a,b), cal_7prb)
        (cpt,cps) = 双线性插值((a,b), 四边形)
        denom = 1 + cpt + cps
        Pt = (p7*(1+cps) + cpt*p_avg) / denom
        Ps = (cps*p7 + p_avg*(1+cpt)) / denom

    ELSE:  // ===== 大角度模式 =====
        (hole1, hole2) = 找最大两孔压力(p1..p6)

        // 尝试 first 孔
        校准_孔1 = 读取大角校准(校准目录, hole1)
        边界_孔1 = 构建大角边界(校准_孔1, hole1)
        (ka1, kb1) = 计算大角kakb(data, hole1)
        sign1 = 射线法((ka1,kb1), 边界_孔1)

        IF sign1 == 0 OR sign1 == 1:
            n = hole1
            (ka, kb) = (ka1, kb1)
            校准 = 校准_孔1
        ELSE:
            // 尝试 second 孔
            校准_孔2 = 读取大角校准(校准目录, hole2)
            边界_孔2 = 构建大角边界(校准_孔2, hole2)
            (ka2, kb2) = 计算大角kakb(data, hole2)
            sign2 = 射线法((ka2,kb2), 边界_孔2)

            IF sign2 == 0 OR sign2 == 1:
                n = hole2
                (ka, kb) = (ka2, kb2)
                校准 = 校准_孔2
            ELSE:
                // 越界处理
                n = hole1
                (ka, kb) = (ka1, kb1)
                边界 = 边界_孔1
                a = 45  // 边界外推 a 默认 45
                IF kb in 边界范围:
                    b = 线性插值(kb, 边界[1])  // 沿 45° 边界
                ELSE:
                    b = 孔角度映射[n]  // 0,60,120,180,240,300

        // 大角度内计算
        网格 = 构建大角四边形网格(校准, n)
        (a_local, b_local) = 四边形内插值_KaKb((ka,kb), 网格)
        四边形 = 定位大角ab四边形((a_local,b_local), 校准, n)
        (cpt,cps) = 大角双线性插值((a_local,b_local), 四边形)

        p_center = data["p"+n]
        p_left   = 左邻孔(data, n)
        p_right  = 右邻孔(data, n)
        p_side   = (p_left + p_right) / 2
        denom = 1 + cpt + cps
        Pt = (p_center*(1+cps) + cpt*p_side) / denom
        Ps = (cps*p_center + p_side*(1+cpt)) / denom

        // === 坐标变换 ===
        θ = a_local
        φ = b_local
        IF |θ| < 89.5:
            tan_a = tan(radians(θ)) * sin(radians(φ))
            tan_b = tan(radians(θ)) * cos(radians(φ))
            a = -degrees(atan(tan_a))
            b =  degrees(atan(tan_b))
        ELSE:
            a = -θ
            b = φ

    // === 速度/马赫数 ===
    ΔP = |Pt - Ps|
    V  = sqrt(2 * ΔP * 287.06 * (t+273.15) / pa)
    ratio = (Pt + pa) / (Ps + pa)
    Ma = sqrt(5 * |ratio^(0.4/1.4) - 1|)

    RETURN { a, b, Pt, Ps, V, Ma }
```

## 14. 异常与边界条件

| 情况 | 处理策略 |
|------|----------|
| Pt < Ps（负压差） | 发出警告，取绝对值计算速度 |
| 压力比 < 0 | 发出警告，取绝对值计算 Ma |
| θ ≥ 89.5°（接近90°） | 跳过 tan 变换，直接用原始值 |
| 分母 1+cpt+cps = 0 | 抛异常，方程无唯一解 |
| (ka,kb) 两点都不在边界内 | 越界处理，a=45° 固定，b 线性外推 |
| 校准文件缺失 | 返回 None，抛异常 |
| JSON 解析失败 | 返回 None |
